# Career Tab

The **Career** tab shows your championship standings and aggregated driver statistics across all championships.

## Championships sub-tab

Each championship is shown as a collapsible section with:

- **Status badge** — Pending, Active, or Finished
- **Points system** — the first few values of the configured points system
- **Driver Standings** — drivers ranked by total points, with wins as tiebreaker
- **Constructor Standings** — shown when constructor scoring is enabled for the championship
- **Rounds** — each round is expandable and shows the sessions it contains, with full result tables

### Round results

Each session within a round shows:
- Session type (Race, Qualify, Practice)
- Track name and date recorded
- Number of drivers
- Full finishing order with fastest lap times
- Points awarded per driver (race sessions only)
- DNF indicators (race sessions only)

## Driver Stats sub-tab

An aggregated stats table across all championships. Click any column header to sort.

| Column | Description |
|---|---|
| **Driver** | Driver name |
| **Races** | Total race starts across all championships |
| **Wins** | First-place finishes |
| **Top 3** | Podium finishes |
| **Top 10** | Top-10 finishes |
| **DNF** | Did-not-finish races |
| **Champ Wins** | Number of championship titles (Finished championships only) |
| **Avg Pos** | Average finishing position across all races |

## PDF export

Click the **Download PDF** button in the Career tab to open your browser's print dialog. All championship sections are expanded automatically so nothing is cut off. The print stylesheet strips the navigation and applies a clean light theme suitable for printing or saving as PDF.
